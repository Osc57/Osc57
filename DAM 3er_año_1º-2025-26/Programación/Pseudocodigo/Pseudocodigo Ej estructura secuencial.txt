Ejercicio 17
Un ciclista parte de una ciudad A a las HH horas, MM minutos y SS segundos. El tiempo
de viaje hasta llegar a otra ciudad B es de T segundos. Escribir un algoritmo que determine
la hora de llegada a la ciudad B.

Iniciar Proceso Ej17
	def hPartida,mPartida,sPartida,sInicial,sFinal,sViaje,hLlegada,sLlegada como entero;
	
	Escribir "Dime la hora de salida: ";
	Leer hPartida;
	
	Escribir "Dime los minutos de salida: ";
	Leer mPartida;

	Escribir "Dime los segundos de salida: ";
	Leer sPartida;
	
	Escribir "Tiempo que has tardado en segundos:";
	leer sViaje;
	
	sInicial <- hPartida * 3600 + mPartida*60 + sPartida;
	
	sFinal <- sInicial + sViaje;
	
	hLlegada <- trunc(sFinal / 3600);
	mLlegada <- trunc((sFinal % 3600) / 60);
	sLlegada <- (sFinal % 3600) % 60;
	
	Escribir "Hora de llegada";
	Escribir hLlegada,":",mLlegada,":",sLlegada;
