Ejercicio 1
Algoritmo que pida dos números e indique si el primero es mayor que el segundo o no.

Iniciar Proceso Ej1
	def n1,n2 como entero;
	
	Escribe "Dime un número: ";
	Leer n1;

	Escribir "Dime otro número: ";
	Leer n2;

	Si n1 > n2 Entonces 
		Escribir "El primer número es mayor"
	SiNo
		Escribir "El primer número no es mayor"
	FinSi
FinPrograma

Ejercicio 5
Escribe un programa que pida un nombre de usuario y una contraseña y si se ha
introducido “pepe” y “asdasd” se indica “Has entrado al sistema”, sino se da un error.

Iniciar Proceso Ej5;
	def usuario, pass como Cadena;
	def Cusuario <- "pepe";
	def Cpass <- "asdasd"

	Escribir "Dime tu usuario: ";
	Leer usuario;

	Escribir "Dime tu contraseña: ";
	Leer pass;

	Si (usuario = Cusuario & pass = Cpass) Entonces
		Escribir "Has entrado al sistema";
	Sino
		Escribir "ERROR: Credenciales incorrectas";
	FinSi;
Fin Proceso;
